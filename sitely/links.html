General

  • papercraft-maker: Papercraft viewer, like an online Pepakura. It can open .pdo files.
  • photopea: Online Photoshop
  • iloveimg: Image resizer, among other image tools.
  • imgonline: Cut an image into equal pieces.
  • lolitahistory: Lolita fashion magazine scans.
  • image-map: Image map code generator.
  • pixel-converter
  • squoosh: Image compressor.

For Webmasters

  • W3Schools: A place to start learning basics of HTML. I may or may not have copied some JavaScript from here... The techniques found here can be questionable at times, though.
  • MDN Web Docs: More in-depth documentation than W3Schools and more up-to-date. Good for referencing.
  • xandra on 32bit.cafe: Resources masterlist for the personal web.
  • petrapixel: Coding tutorials and resources.
  • stackoverflow: Forum for developers, good for specific questions. Usually, I just Google my question and try to find a similar thread on there.
  • wave.webaim: Accessibility checker.
  • webaim: Contrast checker for accessibility.
  • screenfly: Test your site on multiple screen sizes. You could also just use Inspect, but hey theres a website for it, too.
  • scripted: Code snippets.
  • brisray: Webring masterlist.
